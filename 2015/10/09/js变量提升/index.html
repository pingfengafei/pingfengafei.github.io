js变量提升

js变量提升是什么？为什么？
和C++、java、C不同，js没用块级作用域
{
var i = 1;
}
console.log(i); // 1

js变量提升：
1：变量申明提升，定义不提升。
2：函数申明提升，定义不提升。因为函数有表达式形式，和函数形式，2者需要分别对待。
